# EDANE 算法文档

## 1. 名称说明

调研报告第 3 章提出的算法实际名称是 **EDANE**：

**Efficient Dynamic Attributed Network Embedding**

中文可表述为：**基于迭代随机投影与增量学习的动态属性网络表征算法**

本文档在忠实于原报告内容的基础上，对该算法进行清晰的整理，并补充适合实验实现的数学抽象与流程说明。

## 2. 问题定义

给定一个动态属性网络：

- 节点集合 `V`
- 边集合 `E`
- 属性矩阵 `X ∈ R^(N × F)`

目标是学习一个低维嵌入表示：

`Z ∈ R^(N × d)`

使得 `Z` 尽可能保留以下信息：

- 网络拓扑结构信息
- 节点属性相似性信息
- 网络随时间演化的动态信息
- 面向大规模场景的存储效率

四类约束：

1. 十亿级图计算要求算法复杂度尽量接近线性
2. 图发生变化时不能每次都进行全量重训练
3. 结构信息与属性信息需要协同融合
4. 节点向量需要支持轻量化存储

## 3. 核心设计

根据报告第 3 章，EDANE 由四个核心模块组成：

1. 基于稀疏随机投影的迭代初始化模块
2. 基于矩阵扰动思想的动态增量更新模块
3. 基于双曲几何注意力的结构属性融合模块
4. 基于量化压缩的轻量化存储模块

总体设计思想：**高效初始化 + 局部更新 + 自适应融合 + 压缩存储**

## 4. 数学抽象

### 4.1 预处理

对于无向图，先构造邻接矩阵 `A`，再构造对称归一化邻接矩阵：

`S = D^(-1/2) (A + I) D^(-1/2)`

- `D`：度矩阵
- `I`：单位矩阵

对属性矩阵 `X` 做标准化处理，得到 `X̄`。

### 4.2 结构初始化

当前实现采用“两阶段初始化”：

1. 先生成稀疏随机矩阵 `R ∈ R^(N × d)`（三元分布 `{-1,0,1}`）并构造锚点：

`Y_anchor = Σ_(k=0)^q α_k S^k R`

2. 再通过显式目标函数优化：

`J(Y)=||Y-SY||_F^2 + λ||Y-Y_anchor||_F^2`

使用梯度下降与早停得到最终结构嵌入。

### 4.3 属性投影

引入轻量级线性映射使属性信息与结构信息位于同一低维空间：

`H_x = X̄ W_x`，其中 `W_x ∈ R^(F × d)`

### 4.4 双曲注意力融合

很多现实网络具有明显层次结构，双曲空间比欧氏空间更适合表达这类关系。

1. 将欧氏表示映射到 Poincaré 球：`p_s = exp₀(H_s)`，`p_x = exp₀(H_x)`
2. 计算可学习门控：`g = sigmoid(W[p_s | p_x] + b)`
3. 双曲空间加权融合：`p = g ⊗ p_s ⊕ (1−g) ⊗ p_x`
4. 映射回欧氏空间：`Z = log₀(p)`

### 4.5 增量更新

当前实现支持 `ΔV+ / ΔV- / ΔE+ / ΔE-` 与属性更新，更新范围固定为一跳邻域（`k=1`）。

核心更新公式：

`y'_i = y_i + α · f(ΔA_i y_{N(i)})`

其中 `f` 为线性加权聚合（mean），节点删除当前采用 soft deletion。

### 4.6 量化压缩

当前实现采用逐维对称 `int8` 量化，并可选构造 binary 副本：

`q_ij = clip(round(z_ij / s_j), -127, 127)`，其中 `s_j = max_i |z_ij| / 127`

恢复：`z_ij ≈ q_ij × s_j`

同时记录量化误差与压缩比，图结构继续使用 CSR 稀疏格式存储。

## 5. 完整算法流程

**输入**：邻接矩阵 `A`、属性矩阵 `X`、嵌入维度 `d`、传播阶数 `q`

**输出**：浮点型嵌入 `Z`、可选量化嵌入 `Q`

**步骤**：

1. 对图结构和属性进行预处理
2. 构造归一化邻接矩阵 `S`
3. 采样稀疏随机投影矩阵 `R`
4. 计算结构表示 `H_s = Σ α_k S^k R`
5. 计算属性表示 `H_x = X̄ W_x`
6. 通过双曲注意力机制融合 `H_s` 与 `H_x`，得到最终嵌入 `Z`
7. 若需要压缩存储，则对 `Z` 做量化处理
8. 当图发生动态事件时，仅更新受影响节点并重新计算它们的融合表示

## 6. 复杂度分析

| 阶段 | 稠密矩阵 | 稀疏矩阵 |
|------|----------|----------|
| 结构初始化 | O(q·N²·d) | O(q·\|E\|·d) |
| 属性投影 | O(N·F·d) | O(N·F·d) |
| 增量更新 | O(\|U\|·d + local_edges·d) | 同左 |

其中 `|U|` 表示受影响节点数量。

当前代码实现已采用 SciPy CSR 稀疏矩阵后端，邻接矩阵 `A` 和归一化矩阵 `S` 均以压缩稀疏行（Compressed Sparse Row）格式存储与运算，因此实际复杂度遵循"稀疏矩阵"列，内存占用为 O(|E|) 而非 O(N²)。

## 8. 实验原型的定位

这份调研报告本质上是一个研究设计方案，当前代码实现应理解为：

- 基于报告设计思想的忠实原型
- 适合进行中小规模实验验证
- 适合后续写毕业论文中的算法设计与实验章节

它还不是一个真正面向十亿级工业图场景的完整工程系统。

## 9. 建议的实验方案

在第一阶段实验中，可以先使用合成动态属性图进行验证：

1. 为节点生成社区标签
2. 让同社区节点以更高概率连边
3. 围绕各类别中心生成节点属性
4. 在初始图上训练 EDANE
5. 评估节点分类和链接预测效果
   - 节点分类：Macro-F1、Micro-F1
   - 链接预测：Link prediction ROC-AUC、Average Precision (AP)
   - 网络重构：Edge reconstruction ROC-AUC
6. 注入边变化和属性变化
7. 测量增量更新时间与指标变化

## 10. 写论文时的注意点

1. 报告中的算法名称应为 **EDANE**，而不是 EDAME
2. 该算法的设计思想主要受到 RandNE、DAMF、DRLAN、STGNh 等工作的启发
3. 当前代码实现是对报告算法框架的原型化落地
4. 如果要扩展到真正的超大规模图，还需要进一步结合稀疏算子、分布式执行和外存计算机制
