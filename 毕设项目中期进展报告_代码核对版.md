## 一、任务要求（依据任务书与调研）

课题为**高效的超大规模动态属性网络表征方法研究**，核心目标包括：

1. 设计并实现可用于动态属性网络的高效嵌入方法；
2. 在网络结构/属性变化时支持增量更新，减少全量重训练开销；
3. 建立可复现实验流程，完成节点分类、链路预测、重构等指标评估；
4. 在工程上兼顾计算效率与存储开销，支撑后续论文写作与答辩材料产出。

## 二、任务实现方案分解（与当前代码一致）

### （1）算法核心模块实现（`src/edane.py`）

当前 EDANE 代码实现了以下模块：

- **初始化建模**：稀疏随机投影初始化 + 显式迭代优化（`fit()` 内调用 `_compute_structure_embedding()`）。
- **属性表征**：`_compute_attribute_embedding()` 生成属性嵌入。
- **融合策略**：
  - 支持结构/属性融合开关（`use_attr_fusion`）；
  - 支持双曲融合或欧氏融合（`use_hyperbolic_fusion`）；
  - 融合门控含可训练参数（`_train_fusion_gate()`）。
- **增量更新**：`apply_updates()` 支持
  - 节点增加/删除（删除为软删除逻辑）；
  - 边增加/删除；
  - 属性更新；
  - 并以受影响节点及一跳邻域进行局部更新。
- **压缩模块**：支持 int8 量化（`quantize`）与可选二值化（`binary_quantize`），并输出压缩比和误差。

### （2）实验流水线与评估（`src/edane_full_pipeline.py`）

端到端流水线已实现：

1. 数据读取与快照构建（支持 synthetic 与 file）；
2. 模型训练与动态更新（EDANE / DANE / DTFormer 三种模型）；
3. 分类评估（`logreg` 或 `centroid`）；
4. 链路预测指标（AUC/AP）与重构 AUC；
5. 实验产物导出：`summary.json`、`metrics_per_snapshot.csv`、`metrics_curves.svg`、embedding 文件等。

补充：
- 提供 `--no-inc` 消融（每快照全量重训）；
- 提供 `--no-attr`、`--no-hyperbolic` 消融（EDANE 专用）；
- 支持 `--eval-protocol repeated_stratified` 等评估协议。

### （3）数据构建与大规模可跑性（`src/prepare_datasets.py` + pipeline 流式读取）

- `prepare_datasets.py` 支持 OAG 压缩包转换为统一 CSV（`edges/features/labels`，可选 `attr_updates`）。
- `edane_full_pipeline.py` 的 file 模式支持流式读取、时间分桶、分区文件等机制：
  - `iter_csv_dict()` 流式扫描；
  - `build_graph_from_files_streaming()` 分阶段构图；
  - 支持 `--partition-dir`、`--node-selection-mode`、`--time-quantile-sample` 等参数。

这部分说明当前工程重点已从“仅小规模验证”扩展到“面向大规模数据的可跑通能力”。
